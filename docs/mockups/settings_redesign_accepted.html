libli — settings redesign (WS4)

Friendly controls replacing the raw {{ form.as_p }} dropdowns, in the warm-teal identity + top-bar shell. Page 1 = user /settings/ (every user). Page 2 = /settings/institution/ (Platform-Admin only). Each shown light then dark — same markup, two token scopes. Supersedes only the user-settings card of auth-and-settings_accepted.html; the auth + login screens there are unchanged.

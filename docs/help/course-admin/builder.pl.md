# Tworzenie kursu

**Kreator kursu** to miejsce, w którym nadajesz kursowi strukturę i wypełniasz go
lekcjami oraz quizami. Otwórz go z menu **Zarządzaj → wybrany kurs → Kreator**.

## Presety struktury

Każdy kurs korzysta z jednego z czterech presetów struktury, wybieranego w legendzie
kreatora:

- **Płaska** — pojedyncza, płaska lista jednostek, bez grupowania.
- **Rozdziały** — jednostki pogrupowane w rozdziały.
- **Części** — rozdziały pogrupowane w części.
- **Pełna** — części → rozdziały → sekcje → jednostki, najgłębszy układ.

Wybierz najprostszy preset, który pasuje do Twojego materiału; możesz go później
pogłębić, a istniejące jednostki zostaną zachowane.

## Dodawanie jednostek

Użyj przycisku **Dodaj jednostkę**, aby utworzyć *lekcję* (strony z treścią) lub
*quiz* (oceniane pytania). Przeciągaj jednostki, aby zmienić ich kolejność; konspekt
po lewej stronie odzwierciedla to, co widzą uczniowie.

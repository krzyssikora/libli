# Roster management

A group's **roster** — its students and teachers — is edited right inside the
group form. Open a group from **Groups** (or **My groups**) and press **Edit**,
or start a fresh one with **New**.

## Picking students

The student picker is a checkbox list of everyone eligible for the group's
course. Two filters help you find people quickly:

- **Cohort** — a dropdown offering *All cohorts* plus each cohort by name. Pick
  one to narrow the list to that intake.
- **Search by name** — a text box that matches any part of a name,
  case-insensitively, as you type.

A live counter shows **shown / total** while you filter, so you always know how
much of the list the filter is hiding. A separate **Added: N** count tracks your
current selection, with a **(saved: N)** hint when it differs from what was last
saved — a reminder that you have unsaved changes.

## Adding and removing

Tick a student to add them; untick to remove them. Press **Save** to apply: any
student left unticked is removed from the group. Filtering never loses a
selection — every checkbox stays in place behind the scenes, so a student you
selected under one filter is still saved even if a later filter hides them.

The teacher picker works the same way, with just the name search.

## Cohorts are assigned elsewhere

Moving a student *between* cohorts is a separate action, done by a platform admin
on the cohort's own edit page (its **Assign students** list), not from the group
roster. The cohort filter here only *filters* the roster — it never changes which
cohort a student belongs to.

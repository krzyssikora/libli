# Tags & notes

**Notes** and **tags** are your own private annotations on course material. They
are personal to you — never shared with students or other staff — and everyone
with an account can use them, which makes them handy for marking up your own
teaching or study.

## Personal notes

Every content block on a lesson page has a small note handle. When a block has no
notes yet it reads **Add note**; once you have added some it shows a note icon
with the count, for example **(2)**.

- Click the handle to open a popover over the block. Existing notes appear as
  cards, with a composer below for a new one.
- With notes already present, use **Add another note** to append more.
- Each note can be edited or deleted from its card.

Notes stay pinned to the block they belong to, so re-opening a lesson brings your
annotations back exactly where you left them. A saved note deep-links back to
itself, reopening the panel and scrolling to the note.

## Tags on units

Each lesson or quiz has a collapsible **Tags** panel at the top, showing a count
such as **Tags (3)**. Tags are coloured labels you attach to whole units, useful
for grouping material your own way — "revise before exam", "great examples",
whatever helps you.

- Existing tags show as coloured chips, each with an **×** to remove it from the
  unit.
- To add one, type into the **Add a tag…** box (an autocomplete list suggests
  tags you already use) or tick any of your existing tags not yet on this unit,
  then press **Add**.

## Managing your tags

The **My tags** page — reachable from the nav link of the same name — lists all
the tags you have created. From there you can **rename**, **recolour**, or
**delete** any tag. Deleting a tag removes it from every unit it was on. If you
have none yet, the page invites you to open a lesson or quiz and add your first.
